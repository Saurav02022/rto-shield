# Repository agent instructions

The frontend and backend are maintained separately. For each task, read **only** the guide for the area you are changing — there is no need to read both.

| Scope of work | Read this first |
|----------------|----------------|
| **Frontend** — `frontend/`, UI, Next.js app | [`frontend/AGENTS.md`](frontend/AGENTS.md) |
| **Backend** — APIs, database, server code under `backend/` (or whichever directory is the backend root) | [`backend/AGENTS.md`](backend/AGENTS.md) |

For changes confined to the repository root only (CI, shared documentation, tooling) that do not touch an application folder, this `Agent.md` is sufficient. When a change relates to an app, follow that app’s `AGENTS.md`.
